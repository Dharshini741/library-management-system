# Advanced Policy-Driven Library Management System

## Description
A Python-based Library Management System using Tkinter GUI and SQLite database.
Supports book management, student management, issue/return policies, and admin controls.

## Technologies Used
- Python
- Tkinter
- SQLite
- Pillow (for background image)

## Features
- Add / View Books
- Add / View Students
- Issue & Return Books
- Policy-based issuing
- Background image UI

## How to Run
1. Install Python 3.10+
2. Install Pillow:
   pip install pillow
3. Run:
   python library.py
